# Kakasan

Landing page for Raj Sherjeet Singh (Slavik), an aspiring yoga teacher based in Den Haag.

## Website

This repository is published with GitHub Pages from the `main` branch. The site is dependency-free and uses only static HTML, CSS, JavaScript, and local image assets.

## Languages

English is the default language. Visitors can switch the complete interface and page content to Ukrainian; their explicit choice is remembered locally in the browser.

## Privacy

The website has no contact form, cookies, analytics, trackers, external fonts, or third-party scripts. The language preference is stored only in browser `localStorage` and is never transmitted. Email and phone links are assembled in the browser only after a visitor asks to reveal them. This discourages basic scraping, but public contact details cannot be made completely secret on a static website.
