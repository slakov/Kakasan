# Kakasan

Landing page for Raj Sherjeet Singh (Slavik), an aspiring yoga teacher based in Den Haag.

## Website

This repository is published with GitHub Pages from the `main` branch. The site is dependency-free and uses only static HTML, CSS, JavaScript, and local image assets.

## Privacy

The website has no contact form, cookies, analytics, trackers, external fonts, or third-party scripts. Email and phone links are assembled in the browser only after a visitor clicks “Reveal”. This discourages basic scraping, but public contact details cannot be made completely secret on a static website.
